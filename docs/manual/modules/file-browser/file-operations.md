## Context Menu

## Right-click any file to access these options plus

- View Info: Shows detailed metadata including file size, upload date, path, and processing history

## File Info Details

- Name and path in workspace

- File size

- Upload timestamp

- Category

- Processing history (shows if the file was generated by a module)
